What is a living style guide?

It is a methodology for documenting your CSS

A set of components(blocks) which look and feel are defined by the web site styles(CSS)

Thinking of the website design as components not pages.

What is the tool used for generating the style guide?

KSS Node: Automates the generation of the style guide

KSS node uses twig as template engine, which is the Drupal's default one

Why one should consider using a style guide

Developers

  • Avoid code repetition - style guide first approach
  • CMS agnostic
  • Could be used by back-end developers to integrate into CMS

Content Editors

  • Get a visual overview of the components while building pages
